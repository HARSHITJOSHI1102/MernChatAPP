# MernChatAPP

# 💬 Fullstack Chat App

A modern, responsive chat application built with:

- ⚛️ React + Tailwind CSS + Vite (Frontend)
- 🟢 Node.js + Express + MongoDB + Socket.io (Backend)
---
## 🔧 Project Structure
frontend/
├── public/
└── src/
    ├── components/
    │   └── skeletons/
    ├── constants/
    ├── lib/
    ├── pages/
    └── store/

backend/
└── src/
    ├── controllers/
    ├── lib/
    ├── middleware/
    ├── models/
    ├── routes/
    └── seeds/


### Backend Highlights
- 📦 RESTful API
- 🛡 JWT Authentication & Authorization Middleware
- 📁 MongoDB Models: Users & Messages
- 🌩 Real-time Chat using Socket.io
- ☁️ Cloudinary Integration for Profile Uploads

### Frontend Highlights
- ⚡ Vite + React
- 🎨 Tailwind CSS for styling
- 🌙 Dark Mode Support
- 🧠 Zustand for state management
- 🗨 Real-time messaging and chat UI
